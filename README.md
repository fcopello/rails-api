# Barbante's Recommendation API #

This Rails API implements Barbante's recommendation engine do provide a recommendation service to one or more sites.

### Requirements ###

* Ruby version 2.1.0
* Rails version 4.0.2
* MongoDB 2.6.9
* Redis
* Memcached
* [Barbante](https://github.com/hypermindr/barbante)
